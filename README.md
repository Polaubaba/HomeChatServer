# Home Chat Server

I made a chat server, with a public domain name. This do not require google authentication, as we believe Simplicity is the goal here. 
Where anonymous system can easily allow you to chat normally.

I believe we need simplicity back into our lives. 

# SECUIRITY HAZARD ⚠️
Even though it is a very simple and light weight application, just by simply running this on your docker can enable the server. It is quite open and exposed network. 
A simple attack can hamper and compromize the whole system. So basically we need to be cautious.
